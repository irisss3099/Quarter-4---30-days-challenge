This project contains 5 Claude Code Skills for book authors:
- Chapter Outline Generator
- Character Development Assistant
- Plot Consistency Checker
- Research & Fact-Checking Tool
- Grammar & Style Enhancer

Folder: `.claudecode/` contains one folder per skill with `manifest.json` and `handler.md`.

How to use:
1. Place the `.claudecode` folder in your project root.
2. Open VSCode and verify the file tree shows `.claudecode` and all subfolders.
3. Follow your Claude Code environment docs to load or import skills.
4. Test each skill using the example input found in each `manifest.json`.
